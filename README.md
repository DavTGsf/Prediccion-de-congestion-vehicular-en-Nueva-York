# Proyecto de Predicción de Tráfico con Apache Spark

## Descripción
Este proyecto implementa un sistema de análisis y predicción del tráfico vehicular utilizando Apache Spark y Machine Learning (Random Forest Regressor).

El objetivo es analizar patrones de congestión urbana y predecir el volumen de tráfico por hora y zona geográfica.

---

## Arquitectura del sistema

Dataset → HDFS → PySpark ETL → Feature Engineering → Spark MLlib → Evaluación → Visualización

---

## Tecnologías utilizadas

- Apache Spark (PySpark)
- Hadoop HDFS
- Python 3
- Pandas
- Matplotlib
- Spark MLlib (Random Forest)
