# Dataset Source

## Name
Automated Traffic Volume Counts

## Official Source
https://data.cityofnewyork.us/Transportation/Automated-Traffic-Volume-Counts/7ym2-wayt/about_data

## Provider
New York City Department of Transportation (NYC DOT)

## Description
Dataset containing automated traffic volume counts collected across multiple boroughs of New York City.

## Usage in this project
- Distributed processing with Apache Spark
- Feature engineering (hour, borough, rush hour)
- Predictive modeling using Random Forest
